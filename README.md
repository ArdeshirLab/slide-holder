# slide-holder
CAD Files and Writeups for the CosMx-Scan Friendly Slideholder

When preparing slides for use with the CosMx Spatial Molecular Imager (SMI), part of the process involves partitioning tissue samples (which can be stored in different formats, such as FFPE). A common challenge in this step can be properly aligning tissues to a small region of the slide used by the SMI, compounded by difficulties in accurately drawing the region using pens like Sharpies. To assist in accurately drawing these regions for use in tissue slide generation, a slide holder was created using computer-assisted design (CAD). This repository contains the CAD files for the slideholder.

# Why FreeCAD?
Its status as a free software tool with strong community support is why it was used (and why FreeCAD files appear in this repository). Generic 3D model file formats have also been included to allow for use in other software packages and suites (i.e Blender or Autodesk).
